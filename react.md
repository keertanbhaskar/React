- React is JavaScript Library
- React use to make web front-end application
- with react we make SPA(Single App Application)

- Library = A library is a collection of pre-written code that you call when you need it.

        - You are in control of the program flow
        - You decide when and where to use the library
        - It provides specific functionality (e.g., math functions, HTTP requests)

- Framework = A framework provides a structure for your application and controls the flow of  execution.

        - The framework is in control (it calls your code)
        - You plug your code into it at predefined places
        - It defines the architecture and rules
